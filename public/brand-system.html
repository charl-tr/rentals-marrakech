08 — Rayon

Border radius.

Zero. Les cartes, boutons, inputs et images ont des angles droits. C'est un choix de marque : l'architecture marocaine est geometrique, les zelliges sont des lignes droites. Le seul arrondi tolere : les avatars conseillers (cercle).

09 — Principes

Regles.

01
Le contenu est roi.
Les photos des biens sont le premier materiau. Si la photo est mauvaise, aucun design ne la sauve. Investir dans la photographie avant le code.
02
Un accent par ecran.
Le terracotta est rare. Un eyebrow, un CTA, un badge — jamais deux elements terracotta dans le meme viewport. La rareté cree la hierarchie.
03
L'espace parle.
Les marges genereuses sont un signal de luxe. Un site serre fait discount. Les sections respirent avec 112px de padding vertical minimum.
04
Serif pour l'emotion, sans pour l'info.
Playfair Display ne descend jamais en dessous de 18px. Inter ne monte jamais au-dessus de 16px (sauf hero subtitle). La frontiere est nette.
05
Pas de decoration.
Pas d'icones decoratives, pas de gradients colores, pas de bordures fantaisie. Les separateurs sont des lignes beige de 1px. Les ombres sont a peine visibles.
06
Le texte sur image est toujours lisible.
Overlay gradient obligatoire. Drop-shadow sur le texte en dernier recours. Si le contraste n'est pas garanti, le texte sort de l'image.
