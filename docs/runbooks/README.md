# Runbooks

Procédures courtes et exécutables pour les cas ops récurrents.

Format de chaque runbook :
1. **Objectif** en 1 ligne.
2. **Symptômes** (comment on sait qu'il faut ouvrir ce runbook).
3. **Étapes** numérotées, exécutables.
4. **Vérification** que c'est résolu.
5. **Prévention** si applicable.
